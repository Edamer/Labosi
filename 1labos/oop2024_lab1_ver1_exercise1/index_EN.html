In the hr.fer.oop package in the LabArray class, write the method:

public static void increasing(int[] arr)

which will fill the array with randomly selected integers in the following way:

  • the first number will be in the range [1, 10>
  • each subsequent number will be in the range [previous number, 2 * previously generated number>

Example: [9,12,20,20,36], where the first number 9 is generated from [1,10>, then the second number 12 is generated randomly from [9,18>, then 20 is generated from [12,24>, then 20 is generated again from [20,40>, and finally 36 is generated from [20,40>. If another number were to be generated, it would be randomly drawn from the range [36,72>.

You may use the java.util.Random class or the Math.random() method. You may not use any of the other standard classes.

Note: the task is tested with random data and it is possible that there are no errors in one run and errors in another run (non-deterministic). Therefore, it is necessary to pay attention to the edge cases of the task.
