Potrebno je napisati javnu statičku metodu calculateXor i klasi hr.fer.oop.Xor.Xor.Xor koja prima dva stringa koji sadrže nule i jedinice (nije potrebno provjeravati ispravnost ulaznih stringova). Nad stringovima je potrebno napraviti operaciju XOR i vratiti rezultat u obliku stringa bez vodećih nula.

Prototip metode je sljedeći: public static String calculateXor(String s1, String s2);. Nije dozvoljeno koristiti nikakav dodatni import.

PRIMJER:

  • Ulazni stringovi: "10100", "01011". Izlazni string: "11111".
  • Ulazni stringovi: "10", "1000". Izlazni string: "1010".
  • Ulazni stringovi: "110", "100". Izlazni string: "10".
