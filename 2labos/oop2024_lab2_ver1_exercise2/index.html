U zadatku je potrebno napisati klase Task, TaskTag, PriorityTag, DateTag i ParameterTag.

U svim klasama je potrebno i implementirati metode kako su prikazane na dijagramu klasa. Slobodno možete dodati i implementirati dodatne metode, atribute i konstruktore.

Klasa Task sadrži tekst i može sadržavati 0 ili više TaskTag-ova. Prilikom poziva metode toString se vraća tekst na koji su spojeni (konkatenirani) TaskTag-ovi. Sve se odvaja jednim razmakom. Konkretan primjer pogledajte u primjeru korištenja. U Task se može dodati TaskTag metodom add. Task može imati maksimalno 10 TaskTag-ova. Ako se proba dodati 11. TaskTag ništa se ne dodaje u Task.

Svaka vrsta oznaka (TaskTag) se na drugačiji način prikazuje tj. drugi je format zapisa. Primjeri zapisa oznake:

  • TaskTag, name="done" - zapis: @done
  • PriorityTag, name="todo", priority=3 - zapis: @todo(3)
  • ParameterTag, name="repeat", parameter="daily" - zapis: @repeat(daily)
  • DateTag, name="due", year=2020, month=7, day=15- zapis: @due(2020-07-15) (format je yyyy-mm-dd tj. 4 znamenke za godinu, 2 za mjesec i 2 za dan ako se broj može prikazati s manje znamenki potrebno je ispred dodati nule do zadanog broja znamenki).

U klasi DateTag treba paziti da se mogu zapisati samo ispravne vrijednosti. Ako neka vrijednost nije ispravana potrebno je staviti prvu najbližu ispravnu vrijednost. Npr. za mjesec ako u konstruktoru dobijemo 15 onda ga treba postaviti na 12 ili za dan ako je ulazna vrijednost 0 potrebno ju je staviti na 1. Za dan pazite koji mjesec ima koliko dana. Zanemarite prestupne godine.

Primjer korištenja:

Task task = new Task("some task");
System.out.println(task);

task.add(new TaskTag("todo"));
System.out.println(task);

task.add(new PriorityTag("priority", 3));
System.out.println(task);

task.add(new ParameterTag("repeat", "weekly"));
System.out.println(task);

task.add(new DateTag("due", 2020, 5, 1));
System.out.println(task);

task.add(new DateTag("start", 2020, 4, 40));
System.out.println(task);

Očekivani ispis:

some task
some task @todo
some task @todo @priority(3)
some task @todo @priority(3) @repeat(weekly)
some task @todo @priority(3) @repeat(weekly) @due(2020-05-01)
some task @todo @priority(3) @repeat(weekly) @due(2020-05-01) @start(2020-04-30)
