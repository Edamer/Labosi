All classes are in package hr.fer.oop. All classes are public.

You need to write 2 classes: Vehicle and Car.

The Vehicle class cannot be instantiated and has two member variables String name and int yearProduced declared as constants. Write the appropriate constructor, getters and setters, if necessary.

The Car class inherits the Vehicle class and has 2 member variables int[] speed (speed in m/s) and int[] RPM (number of revolutions). The specified parameters (vehicle speed and RPM) were measured while the vehicle was driving with a sampling time of 2 seconds. Write the appropriate constructor and implement the following methods:

  • public float calculateAverageAcceleration() - calculates the average acceleration of the vehicle by calculating the acceleration between two consecutive speed sampling points. It is necessary to subtract the speed (current - previous) and divide by the sampling time (2s). Finally, you need to calculate the average of the calculated accelerations. (Assume that there are at least two or more speed measurements.)
  • public float calculateAverageRPM() - calculates the average RPM value
  • public String calculateEcoValue() - calculates whether the drive is an eco drive. If the average acceleration is between [0.5, 2] and the average RPM is between [2000, 2500] then you need to return "eco", otherwise "non-eco".

Additionally, in the Car class you need to add the following method:

@Override
public String toString() {
return "My car " + getName() + " is produced " + getYearProduced() + " and I am "
+ calculateEcoValue() + " driver.";
}

Example of performance:

Car myPolo = new Car("Polo", 2015, new int[]{15, 20, 33, 33, 38}, new int[] {2000, 2500, 3000, 3200, 2600});
System.out.println(myPolo.calculateAverageAcceleration());
System.out.println(myPolo.calculateAverageRPM());
System.out.println(myPolo.calculateEcoValue());
System.out.println(myPolo.toString());

non-eco
